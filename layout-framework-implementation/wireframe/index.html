Wireframes

I probably went too far beyond a basic wireframe and layout, but I had time to do more and get a more detailed layout.

Normally I would begin with mobile, but I decided to start with Desktop as it seems like the most common screen type for users of WebSTAC.
The layout is largely based on dashboard-items which makes it easy to transition to mobile as I can stack and resize them.
As for the tables, there are several different ways I plan on resizing the information to fit on mobile.
In addition, the navigation drawer will become hidden and viewable with a menu button on mobile.

These designs are created on a 12 column grid that I scaled down to 8 columns and 4 columns for tablet and mobile respectively.
The cards are also aligned to the grid so it is easy to scale for all sizes.
